# Where is the Line? - 이성 친구 경계 테스트

## 프로젝트 개요
- Next.js 16 + TypeScript + Tailwind CSS + Zustand
- 앱인토스(Apps in Toss) 미니앱으로 배포 예정
- WebView 기반 퀴즈 앱

## 기술 스택
- 상태관리: Zustand
- 애니메이션: Framer Motion
- UI: Tailwind CSS + shadcn/ui (향후 TDS 전환 예정)
- 공유: 카카오톡 SDK
